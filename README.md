# Rating Information / Quote Overview

Live link to project: https://mikebars.github.io/rating-information-quote-overview/

## How to use:

Prerequisite dependencies:

- `node` - authored on version 14.8.0
- `yarn` - authored on version 1.22.4

### Before running commands:

Before running other commands, run `yarn install` in the project root to install the project dependencies listed in `package.json`.

### List of common commands:

Run `yarn start` to run the development server on http://localhost:3000.

Run `yarn build` to produce a static build of the project.

Run `yarn lint` to run the project linters.

Run `yarn test` to run the project tests.

Run `yarn jest:coverage` to see test coverage.
